### 3. 리스크 관리

#### 🚨 TSLA 단일 종목 집중 리스크 (최고 심각도)
- TSLA는 머스크 발언 하나에 -15% 급락하는 고변동성 종목
- 2022년 -65% 폭락 전례 존재
- **대응**: 신규 자금은 TSLA 추가 매수 중단, 분산 투자 전환

#### 💱 환율 리스크
- USD/KRW ₩1,471 → ₩1,200대 하락 시 Fidelity 평가액 KRW 기준 -18%
- **대응**: KRW 자산 비중 증대, 환헤지 ETF 활용

#### 📉 카카오뱅크 고가 매수 리스크
- 평균단가 $377.64 vs Fidelity $246.41 — 같은 종목을 훨씬 비싸게 매수
- **대응**: 해당 계좌 추가 매수 중단, 환율 유리 시 Fidelity로 일원화 검토

#### 📊 목표 과대 설정 리스크
- 현재 ₩9,570만 → 2028년 5억 = **31개월 내 5.2배** 성장 요구
- **대응**: 목표 현실화 (2028년 3억, 2030년 7억) 또는 수입 증대 계획 병행

### 4. 날카로운 개선 제안

**① 가장 큰 문제 — 달걀을 한 바구니에**
전 재산이 테슬라 단일 종목. 이건 투자가 아니라 **테슬라에 베팅**하는 것.
자산관리의 기본인 분산이 전혀 되어 있지 않음.

**② 카카오뱅크 계좌 전략 오류**
Fidelity는 $246.41에 사서 +58.61% 수익이지만, 카카오뱅크는 $377.64에 사서 +1.86%.
두 계좌 통합 관리 전략 수립 필요.

**③ 출구 전략(Exit Strategy) 부재**
$750 도달 시 Fidelity 분 일부 익절 → 분산 투자 재원 확보 플랜을 지금 만들어야 함.

**④ 수입원 다각화 미언급**
투자 수익만으로 10억 달성은 현실적으로 어려움.
**월 소득 → 저축 → 투자**의 선순환 구조 구축이 핵심.
